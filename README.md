# Onchain Price Alert System

A professional Web3 reference project that demonstrates how token price monitoring and alert logic can be designed using clean and modular blockchain architecture.

This repository focuses on simulating market price feeds, tracking token value changes, and generating alert notifications in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Monitor token price changes
- Simulate market feed updates
- Trigger price-based alerts
- Track alert history and analytics
- Teach Web3 monitoring architecture

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, price feed simulation, alert engine, history logging, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- Token price simulation engine  
- Configurable alert thresholds  
- Alert history tracking  
- Clean console-based output
